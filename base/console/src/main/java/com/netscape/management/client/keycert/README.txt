The keycert package is not used by Console 4.5 and higher versions. However,
property files in the package are used by legacy plugins which are based on
Console 4.2, 4.1 and 4.0.

Thus, the keycert package, consisting only of property files, MUST be kept
along for Console backward compatibility.
